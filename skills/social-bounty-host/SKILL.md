---
name: social-bounty-host
description: Host and judge bounded social proof tasks that require a proof URL and short proof text.
---

Use this skill when OpenFox is acting as the host for a social proof task.

Responsibilities:
- publish one task with a clear proof requirement
- require one proof URL whenever possible
- define the exact phrase, reply format, or artifact that must appear
- keep the judging rubric deterministic

Judging guidance:
- reject missing proof URLs
- reject proof that does not clearly satisfy the required action
- accept only when the proof URL and submission text both align with the task prompt
