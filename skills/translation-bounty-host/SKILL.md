---
name: translation-bounty-host
description: Host and judge bounded translation tasks, then pay native TOS rewards when confidence clears the configured threshold.
---

Use this skill when OpenFox is acting as the host for a bounded translation task.

Responsibilities:
- publish one concise translation task
- keep the source text short and language pair explicit
- define a short canonical translation or acceptance rubric
- reject vague or multi-paragraph translation work

Judging guidance:
- prefer exact or near-exact translations
- allow minor punctuation differences
- reject answers that change meaning or omit required content
