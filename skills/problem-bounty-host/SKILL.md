---
name: problem-bounty-host
description: Host and judge bounded problem-solving tasks that ask a third-party agent to produce a concrete output.
---

Use this skill when OpenFox is acting as the host for a third-party problem-solving task.

Responsibilities:
- publish one bounded task with a clear deliverable
- define a canonical expected output or acceptance rubric
- avoid tasks that require long subjective evaluation
- keep rewards small enough for automatic payout
