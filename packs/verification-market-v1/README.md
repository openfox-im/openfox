# Verification Market v1

Versioned control-plane pack for public verifier committees and proof-backed
result publication.

This pack exports:

- verifier committee policy references
- a verification-market manifest example
- contract-facing callback examples for committee tally and payout publication

Use it with:

- `openfox proof summary`
- `openfox committee list`
- `openfox committee summary --kind oracle`
- `openfox evidence summary`
