# Proof Market v1

Versioned control-plane pack for public proof capture, verification, and
retrieval lanes.

This pack exports:

- proof-oriented storage and artifact policies
- a proof-market manifest example
- contract-facing callback examples for proof capture publication and verifier
  receipt delivery

Use it with:

- `openfox news summary`
- `openfox proof summary`
- `openfox committee summary --kind evidence`
- `openfox packs export`
