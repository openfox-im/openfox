# Verification Market Flow

Reusable operator template for committee-backed verification and result
publication.

This template is intended for third-party products that want to expose:

- verifier committees
- committee tallies and payouts
- proof-backed result publication

Use it together with:

- `verification-market-v1`
- `openfox committee summary`
- `openfox evidence summary`
- `openfox oracle summary`
