# Proof Market Flow

Reusable operator template for a public proof-capture and proof-verification
deployment.

This template is intended for third-party products that want to expose:

- `news.fetch`
- `proof.verify`
- storage-backed public proof bundle retrieval

Use it together with:

- `proof-market-v1`
- `market-operations-v1`
- `openfox news summary`
- `openfox proof summary`
